# RubySage

A Rails engine, packaged as a gem, that lets you chat with your own codebase.

Drop the helper into any layout and a floating widget appears. Ask "what does this page do?" or "how does donor matching work?" and RubySage answers from your actual code, with citations.

## Status

Pre-release. Under active development. See `PLAN.md` for the architecture and build phases.
